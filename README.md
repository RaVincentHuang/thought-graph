
# Thought Mining for Large Language Model Reasoning

This repository contains the official implementation and dataset for the paper **"Thought Mining for Large Language Model Reasoning"**.

The project introduces the **Thought Graph**, a formal data model designed to capture the intermediate reasoning states ("thoughts") and their underlying data dependencies generated by Large Language Models (LLMs) during multi-step problem-solving. Based on this graph model, we provide a systematic framework for evaluating LLM reasoning capabilities and enhancing their performance through automated pattern mining.

## Repository Structure

The project is modularized into two primary sub-projects. Detailed setup, installation, and usage instructions can be found in the respective `README.md` files within each directory.

```text
.
├── thought/        # Thought Graph construction algorithm
├── mining/         # LLM evaluation, pattern mining, and optimization
├── paper.pdf       # Full text of the paper
└── README.md       # This file

```

### 1. `thought/` (Thought Graph Construction)

This directory contains the core `thought-graph` Python library. It implements the dynamic program analysis and data flow tracking algorithms required to automatically construct Thought Graphs from the execution traces of LLM reasoning programs.

*(Note: This sub-project is managed using Pixi. Please refer to `thought/README.md` for environment setup and testing details.)*

### 2. `mining/` (LLM Testing & Reasoning Enhancement)

This directory contains the experimental framework for evaluating and optimizing LLMs. Key components include:

* **`raw_data/` & `processed_data/**`: Datasets for Game of 24, Blocksworld, and GSM8K tasks.


* **`src/frequent_pattern/`**: Code for mining recurring reasoning patterns from thought graphs.
* **`src/prompt_optimization/` & `src/RL/**`: Implementations of our thought-driven enhancement strategies, utilizing explicit rules extracted from the graphs to improve reasoning via prompt optimization and Reinforcement Learning.



### 3. `paper.pdf`

The complete manuscript detailing the theoretical foundation of the Thought Graph, including the probability properties and the analysis of the ThoughtBuild algorithm.

## Getting Started

To replicate the experiments or apply the Thought Graph framework to your own models:

1. Navigate to the **`thought/`** directory to understand how to instrument reasoning programs and build the graph data model.
2. Navigate to the **`mining/`** directory to execute the evaluation pipelines, run frequent subgraph mining, or train models using the extracted reasoning rules.
